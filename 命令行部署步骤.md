---

## 📋 步骤 1：登录 Vercel（约 1 分钟）

在 PowerShell 中运行：

```powershell
vercel login
```

**操作说明：**
1. 命令运行后，会自动打开浏览器
2. 选择用 **GitHub** 登录（推荐）
3. 授权 Vercel 访问你的 GitHub
4. 授权成功后，回到命令行会显示 "Success! Logged in"

---

## 📋 步骤 2：部署项目（约 2 分钟）

**确保你在项目目录中：**


```powershell
vercel
```

**按提示操作（直接回车使用默认值即可）：**

## 📋 步骤 3：添加环境变量（重要！）

部署完成后，需要添加两个环境变量。

### 方法一：使用命令行（推荐）

```powershell
# 添加第一个环境变量
vercel env add JSONBIN_BIN_ID production
```
- 会提示输入值，**粘贴你的 Bin ID**，然后回车

```powershell
# 添加第二个环境变量
vercel env add JSONBIN_API_KEY production
```
- 会提示输入值，**粘贴你的 Master Key**，然后回车

### 方法二：使用网页（如果命令行不行）

1. 访问 https://vercel.com
2. 找到你的项目 `machao-activation`
3. 点击项目进入详情页
4. 点击 **"Settings"** 标签
5. 点击 **"Environment Variables"**
6. 添加两个变量：
   - `JSONBIN_BIN_ID` = 你的 Bin ID
   - `JSONBIN_API_KEY` = 你的 Master Key
   - Environment: 选择 **Production**（或全选）

---

## 📋 步骤 4：重新部署到生产环境

添加环境变量后，需要重新部署：

```powershell
vercel --prod
```

等待部署完成，会显示生产环境域名。
